> 💡 **Nota:** Este repositório contém uma série de aulas práticas de C++ para iniciantes. Cada aula está organizada em pastas separadas.

---

## 📚 Estrutura do Curso

Cada pasta representa um dia de aprendizagem:

* 📁 `dia-1-hello-world` → Primeiro programa em C++
* 📁 `dia-2-variaveis` → Uso de variáveis
* 📁 `dia-3-if-else` → Decisões no código
* 📁 `dia-4-loops` → Repetições com `for` e `while`

---

## 📌 Como usar este repositório

1. Escolhe uma aula (pasta)
2. Abre o `README.md` dentro da pasta
3. Lê a explicação
4. Executa o código

---

## 🎯 Objetivo

Este projeto foi criado para:

* Ensinar C++ do zero
* Ajudar iniciantes
* Tornar a aprendizagem simples e divertida

---

## 🚀 Como executar os exemplos

```bash
g++ nome_do_ficheiro.cpp -o programa
./programa
```
