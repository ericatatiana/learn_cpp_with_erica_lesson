# Exercício Final - Pratica C++ 

![C++](https://img.shields.io/badge/C%2B%2B-Exercise-orange?style=for-the-badge\&logo=cplusplus)

---

##  Desafio

Agora é a tua vez! 

Cria um programa em C++ baseado no jogo do gato e do rato, mas com novas funcionalidades.

---

## Objetivo

Aplicar os seguintes conceitos:

* Variáveis
* Arrays
* `if/else`
* Loops (`while` ou `for`)
* Funções

---

##  Requisitos do exercício

O teu programa deve:

### 1. Criar pelo menos 3 personagens

* Guardados em um array
* Cada personagem deve ter:

  * Nome
  * Idade

---

### 2. Permitir ao utilizador escolher um personagem

* Validar a escolha

---

### 3. Criar um menu com pelo menos 4 opções

Exemplo:

* Correr
* Comer
* Dormir
* Brincar (NOVO 👀)

---

### 4. Usar um loop

O programa deve continuar até o utilizador escolher sair.

---

### 5. Criar funções

Separar o código em funções, como:

* Mostrar personagens
* Mostrar menu
* Executar ação

---

## 🔥 Desafio EXTRA (para subir nível)

Se quiseres ir além:

*  Adicionar pontuação ao personagem
*  Criar uma nova ação (ex: "Lutar")
*  Mostrar uma mensagem diferente dependendo da escolha
*  Permitir jogar novamente sem reiniciar o programa

---

##  Exemplo de saída esperada

```
Escolhe um personagem: 1

1 - Correr
2 - Comer
3 - Dormir
4 - Brincar
0 - Sair

Opcao: 4

Jerryzinho esta a brincar 😂
```

---

##  Objetivo final

No final deste exercício, deves ser capaz de:

* Criar um programa completo
* Organizar código
* Pensar como programador
